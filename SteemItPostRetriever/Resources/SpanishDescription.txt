Este programa capturará sus posts en su cuenta y los
transferirá a una base de datos especificada por usted.

Antes de usar este programa, ustéd debe haber creado
la base de datos y ejecutado el archivo MySQL.SQL
para crear la tabla que almacenará los resumenes de
sus posts.

Para mostrar los posts en su página web, edite el archivo
SteemitPosts.php cambiando la información de la base de
datos MySQL y subiendo el archivo a su servidor web.
Debe tener PHP instalado en su servidor web para que
el script funcione.

¡Eso es todo!
